## Overview
This repository contains an interactive **Business Intelligence dashboard** built using **Microsoft Power BI** with the Superstore dataset.  
The project focuses on analyzing **sales, profit, quantity, categories, sub-categories, and regional performance** to support data-driven decision-making.

## Tools Used
- Microsoft Power BI  
- DAX (Data Analysis Expressions)  
- Microsoft Excel  
- Business Intelligence & Data Visualization  

## Key Features
- Sales, Profit, and Quantity KPIs  
- Category and Sub-category analysis  
- Region-wise and City-wise performance  
- Sales by region using Map visualization  
- Waterfall chart for regional sales contribution  
- Interactive filters and slicers  

## Use Cases
- Data Analyst / BI portfolio project  
- Academic and final-year project submission  
- Power BI practice and demonstrations  

## Conclusion
This dashboard transforms raw retail data into meaningful insights through effective visual analytics, helping stakeholders understand business performance and trends clearly.
